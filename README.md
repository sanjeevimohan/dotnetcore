# .NET CORE NOTES

# Default access modifiers
| Type | Default |
|---|---|
| namespace | public |
| class | internal |
| inteface | internal |
| enum | public |
| method | private |
| member (class) | private |
| meber (interface) | public |
| constructor (explict) | private |
| constructor (implict) | public |
